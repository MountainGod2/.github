# Security Policy

If you think you have identified a security issue with one of my projects, **do not open a public issue**.

To responsibly report a security issue, please navigate to the Security tab for the repo and click "Report a vulnerability."

![Screenshot of repo security tab showing "Report a vulnerability" button](https://raw.githubusercontent.com/MountainGod2/.github/refs/heads/main/img/image.png)

Be sure to include as much detail as necessary in your report. As with reporting normal issues, a minimal reproducible example will help me address the issue faster.

Thank you.
